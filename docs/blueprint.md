# **App Name**: OmniChat

## Core Features:

- Channel Integration: Connect to WhatsApp, Telegram, Facebook Messenger, and website widgets to create a unified chat interface.
- AI Agent Management: Configure multiple AI providers (OpenAI, DeepSeek, Gemini), create custom agent templates, and assign agents to specific users and channels.
- Real-time Chat Interface: A web-based UI using Socket.IO to display live chat conversations from all integrated channels, with message status updates (sent, delivered, read).
- AI-Powered Auto-Reply: Automate initial responses to user messages based on channel-specific AI agents using a rules-based tool to reason over responses.
- Admin Dashboard: A central dashboard for managing users, channels, AI configurations, website widget settings, and viewing logs.
- Widget Embedding: Embeddable JavaScript widget to integrate live chat functionality directly into websites.
- User Authentication: Secure user sign-up, login, and role-based permission control (admin/agent) using JWT.

## Style Guidelines:

- Primary color: Deep blue (#2962FF) to convey trust and stability.
- Background color: Light gray (#F0F2F5) for a clean and modern feel.
- Accent color: Teal (#00BCD4) for highlighting interactive elements and calls to action.
- Body and headline font: 'Inter' sans-serif font for a modern, machined, objective look. 
- Code font: 'Source Code Pro' for displaying code snippets.
- Use consistent and clear icons for different channels, user roles, and message statuses.
- Subtle animations for loading states and message delivery confirmations.
